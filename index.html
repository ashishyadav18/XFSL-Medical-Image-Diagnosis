XFSL Medical AI

Explainable Prototype-Based Few-Shot Learning Framework for Skin Lesion Classification, OOD Detection, Similarity-Gap Uncertainty Estimation and Grad-CAM Explainability.

GitHub Repository Research Paper

Project Overview

This project was developed as a research dissertation focused on Explainable Few-Shot Learning (XFSL) for medical image diagnosis. The framework combines EfficientNet-B0 feature extraction, prototype-based reasoning, Out-of-Distribution detection, uncertainty estimation and Grad-CAM explainability into a unified clinical decision support system.
